# EFD Website - OSDK React Application

FraudGuard - Enterprise Fraud Detection dashboard built with React, OSDK, and Tailwind CSS.

## Setup

```sh
npm install
npm run dev
```

## Stack
- React 18 + TypeScript
- Palantir OSDK (@osdk/client, @osdk/react)
- Tailwind CSS v4
- Vite
- Recharts (charts)
- Leaflet (maps)
- Framer Motion (animations)
